# 2.4 CDN 与边缘计算

## 配置要点
- HTTP/2/3 启用，多路复用
- 智能路由/就近调度，政务/专网选择内网节点
- 路径缓存规则：`/assets/` 长缓存，`/api/` 不缓存或短缓存
- 压缩：Brotli/Gzip 开启
- 大文件分片/Range 请求支持（视频、离线包）

## 一体化场景
- 地图瓦片/倾斜摄影：CDN 边缘缓存，按区域分 bucket；高并发场景开启回源保护
- 视频流：边缘节点推流/转码；优先 HLS，必要时 WebRTC/低延迟协议
- 构建产物：双构建（现代/兼容），CDN 路径隔离；可灰度发布

## 监控
- 命中率、回源率、延迟分布
- 边缘节点健康与容量
- HTTP 状态码分布（4xx/5xx）
