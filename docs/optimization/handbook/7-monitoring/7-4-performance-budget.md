# 7.4 性能预算

## 预算示例（低空一体化）
- LCP < 2.5s（p75），INP < 200ms，CLS < 0.1
- 首屏 JS < 500KB，CSS < 200KB
- 构建产物 < 50MB，主包 < 1.5MB
- 路由切换 < 800ms，地图首屏 < 2s，视频首帧 < 1.5s

## 执行
- CI/LHCI：预算超限失败（LCP、包体积、未使用代码占比）
- Bundle 预算：按 chunk/依赖设置阈值
- 监控预算：线上 RUM 指标越界报警

## 回归管理
- 版本对比：新旧版本关键指标差异
- 变更评审：每个性能相关 PR 附带指标对比
- 灰度与回滚：不达标时快速回滚或降级
