# 7.1 性能监控体系

## 指标
- Web Vitals：LCP/INP/CLS、TTFB、FCP
- 业务指标：地图首屏、视频首帧、路由切换、接口耗时
- 资源指标：首屏 JS/CSS、错误率、长任务数

## 采集
- `web-vitals` + 自定义埋点（beacon/fetch）
- 路由切换 hook 上报耗时
- 关键接口耗时与错误率
- 设备/网络分布

## 上报策略
- 采样率：默认 10%-20%，核心页面全量
- 批量上报，失败重试

## 看板
- p75/p95 趋势，回归报警
- 版本对比：新旧版本指标差异
- 地域/网络/设备切片

## 响应机制
- 门槛：LCP > 3s、INP > 200ms、CLS > 0.1 报警
- 回滚/灰度：版本回滚或降级
