### 练习 2：地图性能优化
优化地图标注点渲染：
```javascript
// 优化前：一次性渲染所有标注点
markers.forEach(marker => {
  map.addMarker(marker);
});

// 优化后：使用聚合和虚拟化
import MarkerClusterer from '@googlemaps/markerclusterer';

const clusterer = new MarkerClusterer({
  map,
  markers,
  algorithm: new SuperClusterAlgorithm({ radius: 100 }),
});

// 只渲染可视区域内的标注点
map.on('moveend', () => {
  const bounds = map.getBounds();
  const visibleMarkers = markers.filter(m => bounds.contains(m.position));
  updateMarkers(visibleMarkers);
});
```

### 练习 3：建立监控体系
实施完整的监控方案：
```javascript
// 1. 安装依赖
npm install web-vitals

// 2. 采集性能数据
import { onCLS, onFID, onLCP } from 'web-vitals';

function sendToAnalytics(metric) {
  fetch('/api/analytics', {
    method: 'POST',
    body: JSON.stringify(metric),
  });
}

onCLS(sendToAnalytics);
onFID(sendToAnalytics);
onLCP(sendToAnalytics);

// 3. 自定义业务指标
performance.mark('map-load-start');
await loadMap();
performance.mark('map-load-end');
performance.measure('map-load', 'map-load-start', 'map-load-end');
```

## 关键概念

### 优化优先级矩阵

| 优化项 | 影响范围 | 实施难度 | 优先级 | 预期收益 |
|--------|----------|----------|--------|----------|
| 代码拆分 | 全局 | 中 | 高 | 减少 50% 首屏体积 |
| 地图优化 | 地图模块 | 高 | 高 | 提升 3x 渲染性能 |
| 依赖优化 | 全局 | 低 | 高 | 减少 30% 构建体积 |
| 视频优化 | 视频模块 | 中 | 中 | 提升播放流畅度 |
| 内存优化 | 全局 | 高 | 中 | 减少 40% 内存占用 |

```

   ├─ 加载优化
   ├─ 渲染优化

3. 效果验证

4. 持续优化
   ├─ 建立监控体系

```
