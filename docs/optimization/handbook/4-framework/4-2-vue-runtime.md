# 4.2 Vue 3 运行时优化

## 组件级优化
- 精简响应式：拆分 store，避免全局大对象；大数据使用 `shallowRef`/`markRaw`
- 减少重渲染：`v-memo`/`v-once`/`computed` 缓存；避免匿名函数/内联对象
- 正确使用 `key`：列表稳定性，避免重建节点
- 事件缓存：`cacheHandlers` 默认开启（Vite），无需每次创建函数

## 渲染性能
- 虚拟列表：长列表、地图侧栏、告警列表使用虚拟滚动
- 延迟 Hydration/懒组件：非首屏组件懒加载；大组件分片渲染
- Teleport：将全局浮层/弹窗移出重排区域

## 场景实践（低空平台）
- 地图工具条：拆为独立懒加载组件，事件处理 memo 化
- 告警表格：虚拟滚动 + 分页查询；行展开用异步加载
- 视频监控卡片：懒加载播放器 SDK，首帧优先，预渲染占位

## 验证
- Vue Devtools Profiler：渲染次数/耗时
- Performance：查找组件重渲染热点
