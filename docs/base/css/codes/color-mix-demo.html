基于 CSS Color 4 规范的颜色混合算法实现

示例1：白色背景 + 红色(40%) + 绿色(20%)
HTML结构：
<div style="background: white">
  <div style="background: #ff0000; opacity: 0.4">
    <div style="background: #00ff00; opacity: 0.2"></div>
  </div>
</div>
计算过程：
1. 白色背景 + 红色(40%) = #FFC9C9
2. #FFC9C9 + 绿色(20%) = #EEB7AB (Linear RGB空间)
3. 实际浏览器渲染: #EFB7AB

🔍 关键发现

1. 颜色空间的重要性：

  • 浏览器在 Linear RGB 空间 进行 alpha 合成
  • sRGB 空间的计算结果与实际情况差异较大
  • Linear RGB 计算结果与浏览器渲染结果几乎一致

2. 背景色的影响：

  • 必须考虑所有背景层，包括页面背景色
  • 白色背景会让最终结果变亮
  • 黑色背景会让最终结果变暗

3. 多层混合的复杂性：

  • 每一层都会影响最终结果
  • 混合顺序很重要
  • 透明度值会累积影响
⚙️ 技术实现
Alpha 合成公式（CSS Color 4）：
C_out = C_f × α_f + C_b × α_b × (1 - α_f)
α_out = α_f + α_b × (1 - α_f)
sRGB → Linear RGB 转换：
C_linear = C_srgb / 12.92, 当 C_srgb ≤ 0.04045
C_linear = ((C_srgb + 0.055) / 1.055)^2.4, 当 C_srgb > 0.04045
Linear RGB → sRGB 转换：
C_srgb = 12.92 × C_linear, 当 C_linear ≤ 0.0031308
C_srgb = 1.055 × C_linear^(1/2.4) - 0.055, 其他情况
